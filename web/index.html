How to use this page

  1. Select Wi-Fi, scan nearby 2.4 GHz networks, or type the SSID manually.
  2. Enter the password. Leave it empty for an open network.
  3. Use Save to store credentials, Test to validate Wi-Fi, or Connect and restart to enter bridge mode.
  4. Use Profiles later to connect with one click to a saved network.
  5. To reopen configuration mode, hold BOOT until the LED blinks fast, then release and open http://wifi.local.
